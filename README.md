
#  Hacker News - SwiftUI

A beautiful, dark-mode enabled news app. You'll be able to check for current news on hacker news and view on your phone. Design and coding completely done using SwiftUI

# Technologies used

- Xcode
- Swift
- SwiftUI
- Networking
- Public APIs
- JSON Decoding
- JSON Parsing
- WebKit
- Declaritive UI
- UITableView
- MVC Design Pattern
- Navigation
